DigiD

In the DigiD project I was member of the reporting team. I developed queries and csv reports in Ruby on Rails. All reports had unit tests in rspec and where deployed with Capistrano.

For real-time reporting I used rails queries feeding a munin frontend.

BelTel

BelTel is project from the Dutch Tax Office. It is the infrastructure and software for a distributed contact center build around Genesys middleware.

The backend components are developed in Java, the telephony stack is build on Genesys middleware and HiPath telephone systems. The frontend is build with Visual Basic 6.

My roles included frontend / backend tester, administrator for the development and test environment and finally maintener for the front application.
